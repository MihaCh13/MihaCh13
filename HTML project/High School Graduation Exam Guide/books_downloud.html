Учебници по информационни технологии информатика

Python

  • Самоучител за начинаещи в програмирането с езика Python.
  • Навлиза в основите на програмирането, предоставяйки последователно и структурирано ръководство.
  • Обхваща прости команди, изчисления, конзолен вход и изход, работа със среда за разработка (IDE).
  • Конструира по-сложна програмна логика с условни конструкции, цикли, вложени цикли.
  • Декларира и извиква функции, предоставяйки примери и задачи с нарастваща сложност.
  • Предоставя видеоуроци към всяка тема, много практически упражнения и безплатна система за оценяване на решенията.
  • Дава базови познания по програмиране за по-нататъшно обучение като "Python developer".
Изтегли

Java

  • Видео уроци и система за автоматично оценяване на кода
  • Писане на програмен код и решаване на задачи
  • Синтаксис на Java, работа със среда за разработка (IDE)
  • Променливи, оператори, изрази, взаимодействие с конзолата
  • Условни конструкции (if-else, switch-case), цикли (for, while, do-while)
  • Работа с методи - деклариране, извикване, подаване на аргументи, връщане на стойност
Изтегли

Java Script

  • За начинаещи
  • Видео уроци и оценяване на код
  • Синтаксис, променливи, оператори, условия и цикли
  • Функции и техники за решаване на задачи
  • Ръководство за напреднали
  • Обхваща рекурсия, структури от данни, алгоритми
  • Лекции и видеоуроци, принципи на ООП
  • Съвременни технологии - lambda изрази, LINQ заявки
Изтегли

C#

  1. Основи на програмирането
    • Отлично съвременно ръководство за начинаещи в програмирането с езика C#.
    • Запознава читателите с начални умения по програмиране, включително синтаксиса на C#, работа със среда за разработка и основни конструкции.
    • Официален учебник за курса "Programming Basics with C#" в Софтуерния университет (СофтУни).
    • Подготвя читателите за приемния изпит в СофтУни, където програмирането се изучава задълбочено.
    2. Изтегли
  2. Принципи на програмирането
    • Съвременно ръководство, фокусирано върху програмирането и алгоритмите с платформата .NET и C#.
    • Включва уроци по основите на програмирането, структури от данни, алгоритми, ООП принципи, LINQ заявки и други.
    • Авторите провеждат безплатни курсове по програмиране на C# за начинаещи в Софтуерния университет.
    • Предоставя стабилна основа за усвояване на професията C# / .NET софтуерен инженер.
    3. Изтегли

HTML и CSS

  • HTML

    Разглеждани са основите на уеб програмирането, като представени са най-често използваните езици за програмиране в средата на Интернет.
    - Представени са таговете и атрибутите на HTML.
    - Съпоставяне между актуалната версия на езика HTML 4.01 и предстоящата версия HTML 5.

  • CSS

    Основната част разглежда езика HTML, таговете и атрибутите му.
    - Показва стилизиращите възможности на CSS версии 1, 2 и 3.
    - Фокусира се върху форматирането на текстове, списъци, таблици, цветове, фон и възможности за форматиране в колони.
